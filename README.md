# Mini Sales CRM

A simple Customer Relationship Management (CRM) web application built using HTML, CSS, and JavaScript.

## 🚀 Features
- Add new customers
- View customer list
- Edit and delete customers
- Track sales stages (New, In Progress, Closed)
- Sales overview dashboard with statistics

## 💻 Technologies Used
- HTML
- CSS
- JavaScript
- localStorage

## 🌐 Live Demo
https://manasakullegowda-17.github.io/mini-sales-crm/

## 📂 Project Structure
- index.html (Home)
- add_customer.html (Add Customer)
- customers.html (Customer List)
- sales_overview.html (Sales Dashboard)
- script.js (Logic)
- style.css (Styling)
